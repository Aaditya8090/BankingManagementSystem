# Banking Management System

## Overview
A simple desktop banking management application built in Java with a graphical user interface using Java Swing. The system simulates core banking operations like account creation, balance inquiry, deposits, withdrawals, and basic loan handling.

## Features
- Account creation and management
- Balance inquiry
- Deposit and withdrawal operations
- Loan and transaction handling
- GUI interface using Java Swing

## Tech Stack
- Language: Java
- UI: Java Swing
- Concepts: Object-Oriented Programming, Data Structures

## How It Works
1. Clone the repository.
2. Open the project in a Java IDE (e.g., Eclipse, IntelliJ, NetBeans).
3. Compile and run the `Main.java` file to launch the application.
4. Use the GUI to perform banking operations like creating accounts, deposits, withdrawals, and viewing balances.
